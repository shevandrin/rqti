# Compose a root element AssessmentItem of xml task

`create_assessment_item()` creates html structure with AssessmentItem
root element (shiny.tag) for xml qti task description according QTI 2.1

## Usage

``` r
create_assessment_item(object)
```

## Arguments

- object:

  an instance of the S4 object

## Value

A list() with a shiny.tag class
