# Authenticate with LMS

A generic function to handle authentication with a Learning Management
System (LMS).

The method to handle authentication with LMS.

## Usage

``` r
authLMS(object, ...)

# S4 method for class 'LMS'
authLMS(object, ...)
```

## Arguments

- object:

  an instance of the S4 object
  [Opal](https://shevandrin.github.io/rqti/reference/Opal-class.md),
  [LMS](https://shevandrin.github.io/rqti/reference/LMS-class.md)

- ...:

  Additional arguments to be passed to the method, if applicable.
