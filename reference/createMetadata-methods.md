# Create an element of metadata

Create an element of metadata

## Usage

``` r
createMetadata(object)

# S4 method for class 'QtiContributor'
createMetadata(object)

# S4 method for class 'AssessmentItem'
createMetadata(object)

# S4 method for class 'AssessmentTest'
createMetadata(object)
```

## Arguments

- object:

  an instance of the S4 object
  ([QtiContributor](https://shevandrin.github.io/rqti/reference/QtiContributor-class.md),
