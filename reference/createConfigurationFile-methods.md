# Create a configuration file for QTI test

Generic function for creating additional configuration file for the
archive with test

## Usage

``` r
createConfigurationFile(object, output)

# S4 method for class 'AssessmentTest'
createConfigurationFile(object, output)

# S4 method for class 'AssessmentTestOpenOlat'
createConfigurationFile(object, output)
```

## Arguments

- object:

  an instance of the S4 object
  [AssessmentTest](https://shevandrin.github.io/rqti/reference/AssessmentTest-class.md)

- output:

  string, a folder to store an xml configuration files
